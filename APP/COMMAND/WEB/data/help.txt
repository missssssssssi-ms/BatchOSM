BatchOS WEB [ Version 1.0.0.0 ]
 専用のブラウザでWEBを閲覧します。
=基本構文=
 $WEB (-OPEN)(-DEL)
 $WEB -HELP
=引数の概要=
 -OPEN ブラウザを開きます。詳細は、ブラウザの概要をご覧ください。
 -DEL  履歴を削除します。
 -HELP ヘルプを表示します。
  このコマンドは、引数が必須です。
=注意=
 このコマンドは、安全を保障するものではありません。
 一部のPCでは、Webサイトにアクセス出来ない可能性があります。
 閲覧履歴は、手動で削除する必要があります。
 不安定でクラッシュする可能性があります。
=ブラウザの概要=
 このブラウザは、WebView2を使用しています。
 タイトルは、どのWebページを表示しても変更されないため、タイトルで閲覧しているサイトを特定することはできないようになっています。
 一部のサイトは、正常に動作しません。
 一部のPCでは、完全な速度が出ない可能性があります。
 通信は、通常の暗号化のみです。
