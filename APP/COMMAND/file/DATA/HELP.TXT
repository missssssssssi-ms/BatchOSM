File for BatchOS [ Version 1.0.000.0 ]
 このコマンドは、ファイルの表示、変更を行います。
=基本構文=
 $FILE (-OPEN)(-SHOW)<FILENAME>
 $FILE (-VIEW)
 $FILE (-HELP)
=引数の概要=
 -OPEN ファイルを開きます。Widnows既定のプログラムで開きます。
 -SHOW テキストファイルをプロンプトで開きます。ShiftJISでエンコードされている必要があります。(廃止される可能性があります。)
 -VIEW ファイルを表示します。CMDでDIVコマンドを実行します。
 -HELP ヘルプを表示します。
=注意=
 関連付けられていないファイルは、実行できません。
 完全パスは非対応です。CDコマンドを実行して移動してください。
 フォルダーは $FOLDER -HELP を実行して確認してください。
